Details pertaining to shared objects that are referenced by but not changed by a transaction.
This information is considered part of the effects, because although the transaction specifies
the shared object as input, consensus must schedule it and pick the version that is actually
used.


```graphql
union UnchangedSharedObject = SharedObjectRead | SharedObjectDelete
```

#### [<code style={{ fontWeight: 'normal' }}>UnchangedSharedObject.<b>SharedObjectRead</b></code>](/references/sui-api/graphql/reference/objects/shared-object-read) <Badge class="badge badge--secondary" text="object"/> 
> 
>

#### [<code style={{ fontWeight: 'normal' }}>UnchangedSharedObject.<b>SharedObjectDelete</b></code>](/references/sui-api/graphql/reference/objects/shared-object-delete) <Badge class="badge badge--secondary" text="object"/> 
> 
>
