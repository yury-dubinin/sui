An argument to a programmable transaction command.


```graphql
union TransactionArgument = GasCoin | Input | Result
```




### Possible types

#### [<code style={{ fontWeight: 'normal' }}>TransactionArgument.<b>GasCoin</b></code>](/references/sui-api/graphql/reference/objects/gas-coin) <Badge class="badge badge--secondary" text="object"/> 
> 
> 
> Access to the gas inputs, after they have been smashed into one coin. The gas coin can only be
> used by reference, except for with `TransferObjectsTransaction` that can accept it by value.
> 

#### [<code style={{ fontWeight: 'normal' }}>TransactionArgument.<b>Input</b></code>](/references/sui-api/graphql/reference/objects/input) <Badge class="badge badge--secondary" text="object"/> 
> 
> 
> One of the input objects or primitive values to the programmable transaction block.
> 

#### [<code style={{ fontWeight: 'normal' }}>TransactionArgument.<b>Result</b></code>](/references/sui-api/graphql/reference/objects/result) <Badge class="badge badge--secondary" text="object"/> 
> 
> 
> The result of another transaction command.
>
