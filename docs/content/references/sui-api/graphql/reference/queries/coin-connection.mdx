The coin objects that exist in the network.

The type field is a string of the inner type of the coin by which to filter
(e.g. `0x2::sui::SUI`). If no type is provided, it will default to `0x2::sui::SUI`.


```graphql
coinConnection(
  first: Int
  after: String
  last: Int
  before: String
  type: String
): CoinConnection
```
